## Web.py starter  

### Setup

Requires web.py version with python 3+ support.

```shell script
#install virtualenv if not alreadt installed
python -m pip install --user virtualenv

#setup env for project
python -m venv env
source ./env/bin/activate

#install web.py
pip3 install web.py==0.40
```

### Starting the app

```shell script
python main.py
```
